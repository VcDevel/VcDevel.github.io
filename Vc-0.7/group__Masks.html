Masks

Detailed Description

Mask classes are abstractions for the results of vector comparisons.

The actual implementation differs depending on the SIMD instruction set. On SSE they contain a full 128-bit datatype while on a different architecture they might be bit-fields.

Prints the contents of a mask into a stream object.

const Vc::short_m m = Vc::short_v::IndexesFromZero() < 3;
std::cout << m << std::endl;

will output (with SSE): 

m[1110 0000]
Parameters
sAny standard C++ ostream object. For example std::cout or a std::stringstream object.
vAny Vc mask object.
Returns
The ostream object: to chain multiple stream operations.
Note
With the GNU standard library this function will check, whether the output stream is a tty. In that case it will colorize the output.
